## Step 2: Update Property Details

### Update Existing Properties:

1. From the homepage, click on any property (e.g., "Beachfront Paradise Villa")
2. Click the **"Edit"** button (pencil icon)
3. Update these fields:
   - **Title** - Property name
   - **Description** - Property details
   - **Price Per Night** - Daily rate
   - **Location** - Where it's located
   - **Bedrooms, Bathrooms** - Number of rooms
   - **Amenities** - WiFi, Pool, Beach, etc.
4. Click **"Save Changes"**

### Create New Properties:

1. Go to **Admin Dashboard** (click your profile icon â†’ Admin)
2. Click **"+ Add New Property"**
3. Fill in all property details
4. Click **"Create Property"**

---

##  Step 3: Add Real Images

### To Upload Property Images:

1. Click on a property
2. Look for the **"Images"** section
3. Click **"+ Add Image"**
4. Choose image from your computer
5. Click **"Upload"**
6. Save changes

**Image Tips:**
- Use high-quality photos (1200x800px recommended)
- Show the best features of the property
- Include exterior, interior, and special amenities

---

##  Step 4: Customize Branding

### Change App Colors & Theme:

The app has **Light Mode** and **Dark Mode** by default. To customize colors:

1. Contact your developer with color preferences
2. Colors to customize:
   - **Primary Color** - Main brand color
   - **Accent Color** - Highlight color
   - **Background** - Page background

### Change App Name/Logo:

1. Go to Admin Dashboard
2. Look for **"Branding Settings"**
3. Upload your logo
4. Update app name
5. Save changes

---

##  Step 5: Set Up Real Payments

Once you're ready to accept payments:

1. Get a **Stripe account** (stripe.com)
2. Get your API keys from Stripe dashboard
3. Contact your developer to add these keys
4. Payments will start working automatically!

---

##  Step 6: View Bookings & Analytics

### Check Bookings:

1. Click your profile icon (top right)
2. Select **"My Bookings"** or **"Dashboard"**
3. See all recent bookings
4. View booking details
5. Process payments

### View Payments:

1. Go to **Admin Dashboard**
2. Click **"Payments"** tab
3. See all completed transactions
4. Download reports

---

##  Step 7: Manage Users

### Create Admin Users:

1. Go to **Admin Dashboard**
2. Click **"Users"** tab
3. Click **"+ Add User"**
4. Enter email and password
5. Select role: **Admin**, **Host**, or **Guest**
6. Click **"Create"**
